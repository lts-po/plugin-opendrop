# SPR opendrop plugin

*NOTE* in development

Build and start
```sh
$ docker build -t opendrop .
$ docker run --rm -ti --privileged --network host opendrop bash
```

Run inside the container
```sh
$ sudo owl -i <WLAN_IFACE>
$ opendrop find
```
